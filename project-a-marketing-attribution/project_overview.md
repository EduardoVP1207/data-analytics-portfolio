# Project A — Marketing Attribution Analysis

## Business Objective
Understand which marketing channels (email, social media, search, etc.) contribute most to customer conversions and ROI.

## Key Business Questions
1. Which channels drive the highest number of conversions?
2. What is the cost per acquisition per channel?
3. Is there a seasonal pattern or campaign timing effect?

## KPIs
- Conversion Rate (%)
- Cost per Acquisition (€)
- ROI per Channel
- Customer Lifetime Value (CLV)

## Tools & Skills
- **SQL**: query and clean campaign data
- **Python (pandas, matplotlib, seaborn)**: perform analysis and visualize trends
- **Power BI**: create dashboards showing channel performance and ROI

## Dataset
[Kaggle: Marketing Campaign Dataset](https://www.kaggle.com/datasets/rodsaldanha/arketing-campaign)
- ~2,000 customers
- Includes demographics, campaign responses, and spend
- Open license for educational use

## Deliverables
- Clean dataset and analysis notebook (`notebooks/`)
- SQL queries for key metrics (`sql/`)
- Power BI dashboard (`powerbi/`)
- Final report with insights (`docs/`)
