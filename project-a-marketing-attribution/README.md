# Project A - Marketing Attribution
## Initial Insights from Marketing Attribution Project

- Dataset is clean with no missing values or duplicates.
- Average CTR is 3.85%, with some campaigns reaching up to 12.85%.
- Average CPC is $0.79, with outliers up to $2.53 per click.
- Average ROI is 1.18, but with high variance: many campaigns underperform (ROI < 1), while a few are highly profitable (ROI > 10).
- Next steps: channel-specific ROI and CPC analysis, temporal trend analysis, and KPI visualization for decision-making.
