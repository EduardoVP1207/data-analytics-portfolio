# Data Sources

## Project A — Marketing Attribution
- Dataset: [Kaggle: Marketing Campaign Dataset](https://www.kaggle.com/datasets/rodsaldanha/arketing-campaign)
- Description: Customer and campaign-level data including channels, spending, and responses.
- License: Open dataset for educational use.

## Project B — Customer Segmentation
- Dataset: [Kaggle: E-Commerce Data (UK)](https://www.kaggle.com/datasets/carrie1/ecommerce-data)
- Description: Transactional data of an online retailer in the UK with customer IDs, product info, and timestamps.
- License: Open dataset.

## Project C — Churn Analysis
- Dataset: [Kaggle: Telecom Customer Churn](https://www.kaggle.com/datasets/blastchar/telco-customer-churn)
- Description: Telecom customer subscription data with churn labels and service features.
- License: Open dataset.

## Project D — A/B Testing
- Dataset: [Kaggle: A/B Testing E-commerce Dataset](https://www.kaggle.com/datasets/zhangluyuan/ab-testing)
- Description: Control and experiment group results from an e-commerce promotion test.
- License: Open dataset.
